# Инструкция по настройке Telegram бота

## Шаг 1: Создание бота

1. Откройте Telegram и найдите **@BotFather**
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Придумайте имя для бота (например: "Flower Box Orders")
   - Придумайте username (должен заканчиваться на `bot`, например: `flowerbox_orders_bot`)
4. **Скопируйте токен**, который выдаст BotFather (выглядит так: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
   - ⚠️ **ВАЖНО**: Сохраните токен! Он понадобится для настройки в Vercel

## Шаг 2: Получение Chat ID

### Способ 1: Через @userinfobot (РЕКОМЕНДУЕТСЯ - самый простой)

1. Найдите в Telegram бота **@userinfobot**
2. Отправьте ему команду `/start`
3. Бот покажет ваш User ID - **это и есть ваш Chat ID**
4. Скопируйте это число (например: `123456789`)

### Способ 2: Через getUpdates

1. Напишите вашему боту любое сообщение (например, "Привет")
2. Откройте в браузере эту ссылку (замените `YOUR_BOT_TOKEN` на токен из шага 1):
   ```
   https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
   ```
3. В ответе найдите блок с "chat" и "id":
   ```json
   "chat":{"id":123456789,"first_name":"Ваше Имя",...
   ```
4. Число после `"id":` - это ваш Chat ID (например: `123456789`)

### Способ 3: Через @RawDataBot

1. Найдите бота **@RawDataBot**
2. Отправьте `/start`
3. Он покажет всю информацию, включая поле `"id"` - это ваш Chat ID

## Шаг 3: Настройка в Vercel

1. Зайдите на https://vercel.com
2. Откройте ваш проект
3. Перейдите в **Settings** → **Environment Variables**
4. Добавьте две переменные:

   **Переменная 1:**
   - Name: `TELEGRAM_BOT_TOKEN`
   - Value: токен из шага 1 (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
   - Environment: Production, Preview, Development (все три)

   **Переменная 2:**
   - Name: `TELEGRAM_CHAT_ID`
   - Value: Chat ID из шага 2 (только цифры, например: `123456789`)
   - Environment: Production, Preview, Development (все три)

5. Нажмите **Save**
6. Переразверните проект (нажмите **Deployments** → выберите последний → **Redeploy**)

## Проверка работы

1. Заполните форму на сайте
2. Отправьте заявку
3. Проверьте Telegram - должно прийти сообщение с данными заявки

## Если что-то не работает

- Проверьте, что токен правильный (без пробелов, со всеми символами)
- Проверьте, что Chat ID - это только цифры (без кавычек, без @)
- Убедитесь, что вы написали боту хотя бы одно сообщение
- Проверьте, что переменные окружения добавлены для всех сред (Production, Preview, Development)
- Убедитесь, что проект переразвернут после добавления переменных
